什么是组件？

组件 (Component) 是 Vue.js 最强大的功能之一。组件可以扩展 HTML 元素，封装可重用的代码。在较高层面上，组件是自定义元素，Vue.js 的编译器为它添加特殊功能。在有些情况下，组件也可以表现为用 is 特性进行了扩展的原生 HTML 元素。

以上摘自Vue的官网关于组件的介绍。我们的Vue应用是有多个组件构成，组件之间需要有数据传递才能让我们的应用正常运转。

数据传递

父子组件

什么是父子组件？父子组件代表的是组件之间的关系：组件A在它的模板中使用了组件B，这种情况下，组件A就是组件B的父组件，组件B是组件A的子组件。它们之间通信，父组件可能给子组件下发数据，子组件则将其内部要发生的事情告诉父组件。在父子组件下，父组件通过prop给子组件下发数据，子组件通过事件给父组件发送消息。

使用prop传递数据

组件实例作用域是孤立的，所以不能在子组件的模板里直接引用父组件的数据。父组件可以通过prop将数据传递给子组件。
